**Cookiecutter React Django** is a framework for jumpstarting production-ready projects quickly.

-------------------

Powered by `Cookiecutter <https://github.com/cookiecutter/cookiecutter>`_, Cookiecutter React Django combines the capabilities of Django as a backend service with the flexibility and ease of use of React into an opinionated framework that allows you to jumpstart a production-ready web application.

This project is based on a series of tutorials written by `Craig Franklin <https://github.com/cfranklin11>`_ called `Creating an app with Docker Compose, Django, and Create React App <https://dev.to/englishcraig/creating-an-app-with-docker-compose-django-and-create-react-app-31lf>`_
